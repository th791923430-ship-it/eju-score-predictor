# 日本EJU留考大学录取预测与智能推荐系统

这是一个专门为日本留学（EJU留考）学生量身定制的**网页版分数预测与智能择校推荐系统**。系统基于 **2023 - 2025 年近三年的 36,353 条真实历史报考和录取数据** 构建，采用非参数统计概率估算和多维度加权匹配算法，为学生提供客观、科学、高精度的升学评估。

---

## 🌟 系统四大核心亮点

1. **一键智能择校推荐**
   - 根据输入的 EJU 成绩（日语、数学、文综/理综）、托福/托业英语成绩，一键扫描全国 554 所大学、2,216 个专业。
   - 自动将推荐学校归类为：🟢 **保底校**（录取率 >80%）、🟡 **适中校**（录取率 50%-80%）、🔴 **冲刺校**（录取率 20%-50%）。
   - 支持日本大学专属的 **文理双通道** 自动适配。

2. **特定大学学部精确录取率预测**
   - 自由搜索并选择具体目标大学和学部。
   - 系统展示动态彩色仪表盘，并给出精细化的录取概率（%）。
   - 结合学校历史录取门槛，自动展示你的分数在“合格最低分 - 中位数 - 合格最高分”区间中的**横向数轴定位**。

3. **历史录退分值具体分布柱状图**
   - 引入 **Chart.js** 交互式图表技术。
   - 直观展示目标学部在不同分数段（以25分为一个区间）的**实际合格人数**与**不合格人数**的堆叠分布，让录取门槛一目了然。

4. **智能专业顾问诊断建议**
   - 根据你的分数与该学部合格中位数的偏差、标准差，以及英语匹配度，自动生成针对性的**专业诊断意见**，指导你的报考策略。
   - 自动过滤并分类 **国公立 / 私立** 院校及 **S级（顶尖名校）/ A级（一流名校）/ B级（知名大学）/ C级（一般大学）** 梯队。

## 🛠️ 技术栈设计

- **后端 (Backend)**: 
  - `Python 3` + `Flask` 轻量级高性能 Web 框架。
  - `Pandas` + `Numpy` 高性能数据处理与数学计算。
  - **内存级统计预计算引擎**：服务器启动时自动在 1.5 秒内对近 3.6 万条原始数据进行聚合分析（计算各专业的合格中位数、标准差、托福录取均值等），实现 API 请求的毫秒级响应。
  
- **前端 (Frontend)**:
  - `Vue 3` (CDN 渐进式引入) - 零构建成本、高响应式绑定的 SPA 体验。
  - `Tailwind CSS` - 现代、美观、优雅的高级响应式设计，完美适配手机与电脑端。
  - `FontAwesome` - 丰富的图标系统。
  - `Chart.js` - 用于绘制历史录取分数分布图。

## 🚀 快速启动指南

## ☁️ Render 长期固定 URL 部署

如果你希望给别人一个一直可访问的链接，建议直接部署到 Render。部署完成后会得到长期有效的域名（例如 `https://xxx.onrender.com`）。

### 1. 推送代码到 GitHub
- 将当前项目推送到你的 GitHub 仓库。

### 2. 在 Render 创建 Web Service
- 登录 Render，点击 New + -> Web Service。
- 选择你的 GitHub 仓库。
- Render 会自动识别项目根目录下的 `render.yaml`。

### 3. 自动构建与启动
- 构建命令：`pip install -r requirements.txt`
- 启动命令：`gunicorn --workers 1 --threads 4 --timeout 180 --bind 0.0.0.0:$PORT app:app`

### 4. 获取固定访问链接
- 部署成功后，在 Render Dashboard 中复制 Service URL，发给其他人即可长期访问。

### 5. 注意事项
- Render Free 方案在无访问时会休眠，首次唤醒可能需要几十秒。
- 只要服务不删除，URL 会保持不变。

### 1. 安装依赖环境
请确保你的电脑已安装 `Python 3`，然后执行以下命令安装必要的数据处理和 Web 依赖包：
```bash
pip3 install pandas openpyxl Flask
```

### 2. 数据清洗与合并（可选）
项目中已经为您生成了清洗完毕的数据。如果您需要重新合并和处理原始 Excel 表格，可以运行：
```bash
python3 clean_data.py
```

### 3. 启动 Web 预测系统
运行 Flask 服务器主程序：
```bash
python3 app.py
```
启动成功后，控制台会输出：
```text
Loading data and precomputing stats...
Precomputed 1910 school-faculty-track profiles.
 * Running on http://127.0.0.1:5001
```

### 4. 浏览器访问
在浏览器中输入以下地址，即可开启你的 EJU 智能择校之旅：
👉 [**http://127.0.0.1:5001**](http://127.0.0.1:5001)

## 📁 目录文件说明

- [`app.py`](app.py)：Flask 服务器核心逻辑，包含录取率估算算法、一键推荐接口和数据缓存引擎。
- [`templates/index.html`](templates/index.html)：纯前端 SPA 页面，包含 Vue 3 交互逻辑、双向滑动绑定、数据仪表盘、顾问点评及 Chart.js 柱状图。
- [`clean_data.py`](clean_data.py)：数据清洗脚本，自动处理 2023-2025 三年多源、异构的表格列名、编码以及合否字段的规范化。
- [`cleaned_eju_data.csv`](cleaned_eju_data.csv)：清洗完成、可直接加载的高质量 3.6 万条标准 EJU 录取历史数据集。
- [`explore_data.py`](explore_data.py) & [`analyze_scores.py`](analyze_scores.py)：底层探索与分值分布分析工具。
